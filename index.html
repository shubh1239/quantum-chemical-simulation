Variational Quantum Eigensolver (VQE) Prototyping

An end-to-end verification of the $H_2$ ground state using analytical derivation, NumPy, and Qiskit.

1. The Analytical Foundation

Using a minimal basis set mapped via a Jordan-Wigner transformation, the two-qubit Hamiltonian for the Hydrogen molecule simplifies to a combination of Pauli terms. By selecting a particle-conserving ansatz state:

$$\mid\psi(\theta)\rangle = \cos(\theta/2)\mid01\rangle - \sin(\theta/2)\mid10\rangle$$

The expectation value of the energy as a function of the rotation angle $\theta$ maps explicitly to the analytic energy surface equation:

$$E(\theta) = -1.052 + 0.7782\cos(\theta) - 0.182\sin(\theta)$$

Optimizing analytically where $\frac{dE}{d\theta} = 0$ yields $\tan(\theta) = -\frac{0.182}{0.7782}$, localizing the optimal parameter at $\theta \approx 2.911\text{ rad } (166.8^\circ)$ with a theoretical ground state energy of **$-1.851199\text{ Hartree}$**.

2. Classical Matrix Verification (NumPy)

To validate the math before deploying to quantum hardware syntax, a raw matrix representation was constructed in Python. Using scipy.optimize, the classical optimizer successfully converged on the identical global minimum:

  • Optimal Parameter ($\theta$): 2.9118 rad
  • Computed Energy: -1.851199 Hartree
  • Delta vs Exact Diagonalization: $3.56 \times 10^{-11}$ Hartree (well within chemical accuracy)
